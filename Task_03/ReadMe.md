# Плагин Openings

## Назначение
Плагин предназначен для работы с отверстиями в модели Revit.  
Он позволяет просмотреть все отверстия в проекте и, при необходимости, создать в модели линии по габаритам отверстий.

---

## Принцип работы:

1. Находит в текущей модели все отверстия (системные проемы + шахты)
2. Открывает окно. Сверху справа выпадающий список всех отверстий в модели по форме Имя (ID) 
3. Ниже, для выбранного отверстия отображает его параметры: имя/ID элемента//ID хоста/имя хоста/свойство ShaftOpening false или true

Пыталась разобраться: проемы для шахты - OST_ShaftOpening, а вырезание проема - Autodesk.Revit.DB.Opening.
Свойство ShaftOpening, как я поняла, не зависит от категории и = false даже если объект проем для шахты.
Не смогла в ревите создать элемент с ShaftOpening = true, возможно только через апи можно
4. Внизу поле с подсказкой
5. Слева графическое окно. Отображает форму выбранного отверстия

6. Позволяет создать линии модели по BoundingBox отверстия:
   - для выбранного отверстия
   - для всех отверстий
   - только для шахт
